Global Spread of COVID-19

Welcome! This web application was created to show the daily statistics of COVID-19 by a given country. The server was removed after July 16th, hence the stats shows only that date. Map visualizes the situation of the neighbouring countries as well. Start the time lapse representation and observe how the new coronavirus spread across the world from January 22nd until July 16th, 2020. The darker the color, the more infections have been confirmed. Purple color means alarming death rate of the virus.
